2020/4/1
1.if(flag.equals(“on”)) 如果flag为空的话 会报空指针异常
所以通常我们写成 if(“on”.equals(flag))就可以避免空指针异常
要养成一个习惯
2.html checkbox的checked属性问题和value属性问题
对于value属性，复选框的 value 属性值不会显示在用户界面中。value 属性用于表单数据的提交（只有选中的复选框才会传递数据到服务端，如果什么都没选则不传递复选框数据）。
只要出现了checked=””，不管等于什么，可以等于checked=”checked”,甚至checked=”false”，其checkbox的属性checked都是true,在alert中输出其checked属性都是true.
